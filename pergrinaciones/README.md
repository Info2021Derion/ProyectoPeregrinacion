
# Plataforma de Eventos Demo

Demo MVP de:
- inscripción pública
- panel admin
- control de cupos
- pagos manuales
- credenciales PDF
- archivos descargables
- roles y configuración

## Recomendado para publicar gratis

### Frontend
- Vercel

### Base de datos
- Supabase

## Instalar

```bash
npm install
npm run dev
```

## Variables de entorno

Crear `.env.local`

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```
